〔Ⅲ〕 次の(a)～(h)の各文の空欄に入れるのに最も適当な語(句)を1～4の中から一つずつ選び，その番号を記入せよ。

(a) Hurry up, (　　) you will miss the train.
　　1. and
　　2. or
　　3. but
　　4. so

(b) I haven't seen her (　　) she moved to another city.
　　1. when
　　2. before
　　3. since
　　4. until

(c) (　　) I like her, I sometimes find her difficult to work with.
　　1. Because
　　2. Although
　　3. If
　　4. Unless

(d) The soup was (　　) hot that I burned my tongue.
　　1. very
　　2. too
　　3. much
　　4. so

(e) Write down his address (　　) you forget it.
　　1. after
　　2. before
　　3. while
　　4. because

(f) (　　) you have finished reading the book, please return it to the library.
　　1. Although
　　2. Once
　　3. Unless
　　4. While

(g) She speaks (　　) French and German fluently.
　　1. either
　　2. neither
　　3. both
　　4. not only

(h) I'll lend you my car (　　) you promise to drive carefully.
　　1. because
　　2. although
　　3. unless
　　4. provided

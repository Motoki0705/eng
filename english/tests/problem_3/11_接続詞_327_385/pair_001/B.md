〔Ⅲ〕 次の(a)～(h)の各文の空欄に入れるのに最も適当な語(句)を1～4の中から一つずつ選び，その番号を記入せよ。

(a) Study harder, (　　) you will pass the exam.
　　1. or
　　2. but
　　3. and
　　4. so

(b) I can speak neither French (　　) German.
　　1. or
　　2. and
　　3. but
　　4. nor

(c) (　　) it rains tomorrow, the event will be held indoors.
　　1. Unless
　　2. If
　　3. Although
　　4. Because

(d) It was (　　) a difficult problem that no one could solve it.
　　1. very
　　2. so
　　3. such
　　4. too

(e) I left early (　　) I could catch the first train.
　　1. because
　　2. so that
　　3. unless
　　4. although

(f) (　　) you like it or not, you have to follow the rules.
　　1. If
　　2. Whether
　　3. Unless
　　4. When

(g) He kept trying (　　) he finally succeeded.
　　1. until
　　2. before
　　3. since
　　4. while

(h) (　　) tired she was, she continued working.
　　1. Although
　　2. Because
　　3. However
　　4. Whatever

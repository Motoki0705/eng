(a) My plan is (　　) my homework before I go out tonight.
　　1. finish
　　2. to finish
　　3. finishing
　　4. finished

(b) I am very happy (　　) that you passed the examination.
　　1. hear
　　2. hearing
　　3. to hear
　　4. heard

(c) The problem was (　　) for him to solve without any help.
　　1. enough difficult
　　2. difficult enough
　　3. so difficult
　　4. too difficult

(d) He worked very hard, only (　　) that his effort was in vain.
　　1. finding
　　2. found
　　3. to find
　　4. find

(e) The heavy rain prevented us from going out, (　　) say nothing of playing tennis.
　　1. not to
　　2. to
　　3. as to
　　4. so as to

(f) It is kind (　　) you to help me with my luggage.
　　1. for
　　2. of
　　3. to
　　4. with

(g) I heard someone (　　) my name in the distance.
　　1. called
　　2. to call
　　3. call
　　4. calling

(h) All you have to do is (　　) your best in every situation.
　　1. do
　　2. doing
　　3. done
　　4. to doing

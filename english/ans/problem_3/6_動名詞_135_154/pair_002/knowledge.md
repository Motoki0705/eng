# 動名詞 pair_002 - 習得すべき知識

## 📚 このペアで学ぶ文法事項

### 1. 動名詞の意味上の主語 (No. 136)

```
所有格/目的格 + doing「（人）が〜すること」
```

**例**: I don't mind **his/him coming** with us.
（彼が一緒に来ることは構わない）

**例**: Do you remember **my/me saying** that?
（私がそう言ったのを覚えていますか）

⚠️ フォーマルな文では**所有格**、カジュアルでは**目的格**も可。

> 🎩 **フォーマル vs カジュアルの使い分け**
> | 場面 | 形 | 例 |
> |-----|-----|-----|
> | 論文・ビジネス文書 | 所有格 | his coming |
> | 日常会話 | 目的格 | him coming |
> 
> ネイティブも普段は目的格を使うことが多いですが、試験では所有格が「正式」とされることが多いので注意！

---

### 2. 動名詞の否定 (No. 138)

```
not doing「〜しないこと」
```

**例**: I regret **not studying** harder.（もっと勉強しなかったことを後悔している）

⚠️ not の位置は動名詞の**直前**。

> ⚡ **not の位置が命！**
> ```
> ⭕ not doing（〜しないこと）
> ✕ doing not（あり得ない形）
> ```
> 不定詞の否定（not to do）と同じパターン。否定語は常に動詞形の直前に置く！

### 3. 慣用表現（追加）

#### There is no sense (in) doing (No. 141)
```
There is no sense (in) doing「〜しても意味がない」
```

**例**: **There is no sense in** waiting for him.

> 🧠 **sense = 「意味」「感覚」**
> make sense（意味をなす）と同じ単語。「〜しても意味がない」という表現は、ビジネスで無駄な会議を批判するときにも使えます！

#### be accustomed to doing (No. 144)
```
be accustomed to doing「〜することに慣れている」
= be used to doing
```

**例**: He **is accustomed to speaking** in public.

> 📚 **accustomed はフォーマル版！**
> be used to doing と同じ意味ですが、accustomed の方が堅い表現。ビジネス文書や公式な場面で使われます。語源はラテン語の "consuetudo"（習慣）。

#### object to doing (No. 145)
```
object to doing「〜することに反対する」
```

**例**: I **object to being** treated like a child.
（子供扱いされることに反対だ）

> 🏛️ **法廷でよく聞く表現！**
> "I object!"（異議あり！）は法廷ドラマでおなじみ。object to は「〜に対して異論を唱える」という強い反対表明です。

---

### 4. be busy (in) doing (No. 150)

```
be busy (in) doing「〜するのに忙しい」
```

**例**: She **is busy preparing** for the party.

> 💼 **なぜ in が省略できる？**
> もともとは "busy in doing" でしたが、現代英語では in の省略が一般的。spend time (in) doing や have difficulty (in) doing と同じパターンです。

---

### 5. It goes without saying that ... (No. 152)

```
It goes without saying that ...「〜は言うまでもない」
```

**例**: **It goes without saying that** health is important.

> 🗣️ **直訳すると面白い！**
> 「それは言われずに行く」→「言わなくても通じる」→「言うまでもない」
> 
> 類似表現：
> - Needless to say, ...（言うまでもなく）
> - Obviously, ...（明らかに）

### 6. 復習項目

#### look forward to doing (No. 142)
```
look forward to doing「〜を楽しみにする」
```

#### be used to doing (No. 143)
```
be used to doing「〜に慣れている」
```

#### when it comes to doing (No. 146)
```
when it comes to doing「〜のことになると」
```

#### What do you say to doing? (No. 147)
```
What do you say to doing?「〜するのはどう？」
```

> 📝 **「to + doing」パターン総まとめ**
> 以下の to は**前置詞**なので、後ろは動名詞！
> | 表現 | 意味 |
> |-----|-----|
> | look forward to doing | 〜を楽しみにする |
> | be used to doing | 〜に慣れている |
> | be accustomed to doing | 〜に慣れている |
> | object to doing | 〜に反対する |
> | when it comes to doing | 〜のことになると |
> | What do you say to doing? | 〜するのはどう？ |
> | devote oneself to doing | 〜に専念する |
> | be committed to doing | 〜に専念している |

#### have difficulty/trouble (in) doing (No. 148, 149)
```
have difficulty/trouble (in) doing「〜するのに苦労する」
```

#### There is no doing (No. 151)
```
There is no doing「〜できない」
```

#### feel like doing (No. 153)
```
feel like doing「〜したい気がする」
```

#### on doing (No. 154)
```
on doing「〜するとすぐに」
```

# 動詞の語法 pair_001 - 習得すべき知識

## 📚 このペアで学ぶ文法事項

### 1. 動名詞のみを目的語にとる動詞

#### mind doing (No. 515)
```
mind doing「〜することを気にする/嫌がる」
```

**例**: Would you **mind opening** the window?

⚠️ Would you mind ...? への返答：
- 「いいですよ」→ No, not at all. / Of course not.
- 「嫌です」→ Yes, I do mind.

#### consider doing (No. 520)
```
consider doing「〜することを検討する」
```

**例**: I'm **considering moving** to Tokyo.

---

### 2. 不定詞のみを目的語にとる動詞

#### decide to do (No. 523)
```
decide to do「〜することを決める」
```

**例**: He **decided to study** abroad.

---

### 3. 動名詞と不定詞で意味が変わる動詞

#### stop doing vs stop to do (No. 519)
```
stop doing「〜することをやめる」
stop to do「〜するために立ち止まる」
```

**例**: He **stopped smoking**.（タバコをやめた）
**例**: He **stopped to smoke**.（タバコを吸うために立ち止まった）

#### remember doing vs remember to do (No. 529, 530)
```
remember doing「〜したことを覚えている」（過去の行為）
remember to do「忘れずに〜する」（未来の行為）
```

**例**: I **remember meeting** him before.（以前会ったことを覚えている）
**例**: **Remember to lock** the door.（忘れずにドアに鍵をかけて）

#### forget doing vs forget to do (No. 531, 532)
```
forget doing「〜したことを忘れる」（過去の行為）
forget to do「〜することを忘れる」（未来の行為）
```

**例**: I'll never **forget visiting** Paris.（パリを訪れたことを忘れない）
**例**: I **forgot to bring** my umbrella.（傘を持ってくるのを忘れた）

#### try doing vs try to do (No. 534)
```
try doing「試しに〜してみる」（実際にやってみる）
try to do「〜しようとする」（努力する）
```

**例**: **Try eating** this cake.（試しに食べてみて）
**例**: I **tried to open** the door, but it was locked.（開けようとしたが無理だった）

#### mean to do (No. 535)
```
mean to do「〜するつもりである」
```

**例**: I didn't **mean to hurt** your feelings.

---

### 4. 使役動詞 (No. 545, 546)

#### make A do (No. 546)
```
make A do「Aに（強制的に）〜させる」
```

**例**: The teacher **made** the students **study** hard.

⚠️ 受動態では **to** が必要：be made **to** do

#### let A do (No. 545)
```
let A do「Aに〜させてやる、〜することを許す」
```

**例**: Please **let** me **know** if you need help.

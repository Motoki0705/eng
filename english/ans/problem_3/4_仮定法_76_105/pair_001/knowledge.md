# 仮定法 pair_001 - 習得すべき知識

## 📚 このペアで学ぶ文法事項

### 1. 仮定法過去 (No. 76)

#### 現在の事実に反する仮定
```
If S + 過去形 ..., S + would/could/might + 原形 ...
「もし〜なら、...だろうに」
```

**例**: **If** I **were** rich, I **would buy** a big house.
（もし私がお金持ちなら、大きな家を買うのに → 実際は金持ちではない）

⚠️ be動詞は主語に関わらず **were** を使うのが正式。

> 🤔 **なぜ「過去形」なのに現在の話？**
> 仮定法の「過去形」は時間の過去ではなく、**「現実からの距離」**を表しています。過去形を使うことで「これは現実じゃないよ」と示すのです。日本語の「もし〜だったら」と似た感覚ですね。言語学では「距離を置く」ための文法手段と呼ばれています。

---

### 2. 仮定法過去完了 (No. 80)

#### 過去の事実に反する仮定
```
If S + had + 過去分詞 ..., S + would/could/might + have + 過去分詞 ...
「もし〜だったなら、...だっただろうに」
```

**例**: **If** I **had studied** harder, I **would have passed** the exam.
（もっと勉強していたら、試験に受かっていただろうに → 実際は勉強しなかった）

> 💭 **歴史のifを考える**
> "If Napoleon had won at Waterloo..."（もしナポレオンがワーテルローで勝っていたら...）
> 歴史好きが大好きな「もしも」の議論は、まさに仮定法過去完了！過去に戻って「別の選択をしていたら」と考える、人間ならではの思考実験です。

---

### 3. S wish + 仮定法 (No. 85)

#### 現在の願望（仮定法過去）
```
S wish + S' + 過去形「〜であればいいのに」
```

**例**: I **wish** I **were** taller.（もっと背が高ければいいのに）

#### 過去の願望（仮定法過去完了）
```
S wish + S' + had + 過去分詞「〜だったらよかったのに」
```

**例**: I **wish** I **had studied** harder.（もっと勉強していたらよかった）

> 🎵 **名曲で覚える wish**
> ビートルズの "I wish I was a child again"、スティービー・ワンダーの "I Wish" — 仮定法を使った名曲は数えきれません。「叶わない願い」を歌うときは自然と仮定法になるのです。

---

### 4. It is time + 仮定法過去 (No. 90)

```
It is time + S + 過去形「もう〜してもいい頃だ」
It is high time + S + 過去形「とっくに〜すべきだ」
```

**例**: It is time you **went** to bed.（もう寝る時間だ）
**例**: It is high time we **left**.（もう出発すべき時間だ）

> ⏰ **「high time」の強調**
> "high" がつくと「もうとっくに過ぎてるよ！」という焦りが加わります。親が子供に "It's high time you cleaned your room!" と言ったら、かなり怒っている証拠。

---

### 5. if it were not for A (No. 92)

#### 現在の事実に反する仮定
```
if it were not for A = without A = but for A
「Aがなければ」
```

**例**: **If it were not for** water, we could not live.
（水がなければ、生きられないだろう）

#### 過去の事実に反する仮定
```
if it had not been for A「Aがなかったら」
```

**例**: **If it had not been for** your help, I would have failed.
（あなたの助けがなかったら、失敗していただろう）

> 🌍 **3つの言い方を制覇！**
> | 現在 | 過去 |
> |-----|-----|
> | If it were not for A | If it had not been for A |
> | Without A | Without A |
> | But for A | But for A |
> 
> どれも同じ意味ですが、文章のリズムや堅さで使い分けます。エッセイでは variety のために交互に使うとGood！

---

### 6. if の省略と倒置 (No. 95)

```
If S were ... → Were S ...
If S had done ... → Had S done ...
If S should do ... → Should S do ...
```

**例**: **Were I** rich, I would travel around the world.
（= If I were rich）

**例**: **Had I** known the truth, I would have told you.
（= If I had known the truth）

> 📚 **文学的で格調高い！**
> if の省略と倒置は、フォーマルな文書や文学作品で使われます。"Had I but known..."（もし知っていたなら...）は小説や詩でよく見かける表現。入試の英作文で使えば「おっ、この受験生やるな」と思わせる効果も！

---

### 7. Should S do ... (No. 98)

#### 万一の仮定
```
If S should do ..., S' + 命令形/will/would + ...
「万一〜したら」
```

**例**: **If you should** see him, please tell him to call me.
（万一彼に会ったら、電話するよう伝えてください）

⚠️ 実現の可能性が低いことを示す。

---

### 8. 副詞句による仮定 (No. 101)

```
with A / without A / but for A で仮定を表す
```

**例**: **With** a little more money, I could buy it.
（もう少しお金があれば、買えるのに）

**例**: **Without** your help, I could not have done it.
（あなたの助けがなければ、できなかっただろう）

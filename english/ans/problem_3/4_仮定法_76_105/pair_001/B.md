## 詳細な解説

### (a) If she (   ) here now, she would help us.
**解答**: ③ were

**Next Stage参照**: No. 76 - 仮定法過去（従節）

**解説**:
- 仮定法過去の条件節では動詞は過去形
- now があるので現在の仮定

---

### (b) If we had left earlier, we (   ) the train.
**解答**: ④ would have caught

**Next Stage参照**: No. 80 - 仮定法過去完了（主節）

**解説**:
- 仮定法過去完了の主節は would have + 過去分詞
- 「電車に乗れていただろう」

---

### (c) She wishes she (   ) taller.
**解答**: ③ were

**Next Stage参照**: No. 85 - S wish + 仮定法過去

**解説**:
- S wish + S' + 仮定法過去
- be動詞は were

---

### (d) It is about time you (   ) looking for a job.
**解答**: ② started

**Next Stage参照**: No. 90 - It is time + 仮定法過去

**解説**:
- It is about time + S + 仮定法過去
- 「そろそろ...してもよい時間だ」

---

### (e) (   ) it not for his support, I could not continue my studies.
**解答**: ③ Were

**Next Stage参照**: No. 92 - if it were not for A

**解説**:
- If it were not for A → Were it not for A
- if を省略して倒置

---

### (f) (   ) he honest, I would trust him completely.
**解答**: ② Were

**Next Stage参照**: No. 95 - if の省略倒置

**解説**:
- If he were honest → Were he honest
- 仮定法過去の倒置形

---

### (g) (   ) there be any problem, contact me immediately.
**解答**: ④ Should

**Next Stage参照**: No. 98 - Should S do...

**解説**:
- If there should be... → Should there be...
- 命令文が主節

---

### (h) (   ) more time, I could do a better job.
**解答**: ① With

**Next Stage参照**: No. 101 - 副詞句 with A に仮定の意

**解説**:
- with A「もしAがあれば」
- 仮定法過去と共に使用

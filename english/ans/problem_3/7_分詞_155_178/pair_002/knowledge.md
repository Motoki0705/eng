# 分詞 pair_002 - 習得すべき知識

## 📚 このペアで学ぶ文法事項

### 1. 分詞の後置修飾（復習・発展）

#### 現在分詞句の後置修飾 (No. 156)
```
名詞 + doing ...「〜している（名詞）」
```

**例**: Look at the boy **running** over there.
（向こうで走っている少年を見て）

#### 分詞が他の語句を伴っている場合 (No. 158)
```
分詞が目的語や副詞を伴う場合は後置修飾
```

**例**: the letter **written in French**（フランス語で書かれた手紙）
**例**: the man **sitting next to me**（私の隣に座っている男性）

> 🎯 **関係代名詞との書き換え**
> | 分詞 | 関係代名詞 |
> |-----|----------|
> | the letter written in French | the letter **which was** written in French |
> | the boy running over there | the boy **who is** running over there |
> 
> 分詞は関係代名詞節の「圧縮版」！長文読解で展開して理解すると楽になります。

---

### 2. There be S doing/done（復習）(No. 159, 160)

```
There is/are + S + doing「Sが〜している」
There is/are + S + done「Sが〜されている」
```

**例**: **There were** many people **waiting** in line.
**例**: **There was** a letter **left** on the desk.

> 📸 **「その場の状況を切り取る」表現**
> There構文 + 分詞は、映画のオープニングシーンのように「ある状況」を描写するのに最適。小説の冒頭でもよく使われます。

---

### 3. 知覚動詞 + O + doing/done (No. 162, 163, 164)

#### hear A doing (No. 164)
```
hear A doing「Aが〜しているのを聞く」
```

**例**: I **heard** someone **calling** my name.
（誰かが私の名前を呼んでいるのを聞いた）

#### 目的語と受動関係（過去分詞）(No. 163)
```
知覚動詞 + O + done「Oが〜されるのを...」
```

**例**: I saw the window **broken** by the boy.
（窓が少年に割られるのを見た）

> 🔊 **doing vs done の判断法**
> 目的語が「する側」→ doing（能動）
> 目的語が「される側」→ done（受動）
> 
> - I heard him singing.（彼が歌っている＝彼は歌う側）
> - I heard my name called.（名前が呼ばれた＝名前は呼ばれる側）

---

### 4. make oneself heard (No. 168)

```
make oneself heard「自分の声を届かせる」
```

**例**: I couldn't **make myself heard** above the noise.
（騒音で自分の声を届けられなかった）

> 🎤 **コンサート会場でよくある状況！**
> ライブハウスの大音量の中で友達に話しかけても聞こえない...まさにこの状況が "I couldn't make myself heard" です。

### 5. 分詞構文（発展）

#### 分詞構文の基本 (No. 169)
```
Doing ..., S + V「〜して、SはVする」
```

**例**: **Hearing** the news, she burst into tears.
（その知らせを聞いて、彼女は泣き出した）

> 🎭 **分詞構文の5つの意味**
> 分詞構文は文脈によって様々な意味を持ちます：
> 1. **時**：〜するとき（When ...）
> 2. **理由**：〜なので（Because ...）
> 3. **条件**：もし〜なら（If ...）
> 4. **譲歩**：〜だけれども（Although ...）
> 5. **付帯状況**：〜しながら（while ...）
> 
> 文脈で判断するのがポイント！

#### 完了分詞構文（復習）(No. 171)
```
Having done ..., S + V「〜してから、SはVする」
```

**例**: **Having lived** in London for ten years, he speaks English well.

> 🌍 **経験を語る定番表現**
> 「10年住んでいたから英語がうまい」—原因と結果を一文で表現。自己紹介やプレゼンで使える便利な構文！

#### 独立分詞構文（復習）(No. 172)
```
主語 + doing ..., S + V「（主語が）〜して、SはVする」
```

**例**: **Weather permitting**, we will go on a picnic.
（天気が許せば、ピクニックに行く）

> ☀️ **慣用表現として覚えよう**
> - Weather permitting（天気が良ければ）
> - All things considered（全てを考慮すると）
> - Strictly speaking（厳密に言えば）
> - Generally speaking（一般的に言えば）
> 
> これらは独立分詞構文が慣用句化したもの！

#### 受動態の分詞構文（復習）(No. 173)
```
(Being) done ..., S + V「〜されて、SはVする」
```

**例**: **(Being) surrounded** by mountains, the town is quiet.

> 🏔️ **地理の説明に便利！**
> "Surrounded by mountains, the town is..."
> 旅行ガイドや地理の説明でよく見かける表現。Being はほぼ省略されます。

### 6. 慣用的分詞構文（追加）(No. 176)

| 表現 | 意味 |
|-----|-----|
| granting (that) | 〜を認めるとしても |
| supposing (that) | もし〜ならば |
| provided (that) | もし〜ならば |
| given A | Aを考慮すると |
| talking of A | Aと言えば |
| all things considered | すべてを考慮すると |

**例**: **All things considered**, he did well.
（すべてを考慮すると、彼はよくやった）

---

### 7. with A done / with A 形容詞（復習）(No. 177)

```
with + A + doing「Aが〜しながら」
with + A + done「Aが〜された状態で」
with + A + 形容詞「Aが...の状態で」
```

**例**: She was listening **with her eyes open**.（目を開けて聞いていた）
**例**: He stood **with his hands in his pockets**.（ポケットに手を入れて立っていた）

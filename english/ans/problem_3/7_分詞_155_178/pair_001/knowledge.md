# 分詞 pair_001 - 習得すべき知識

## 📚 このペアで学ぶ文法事項

### 1. 名詞を修飾する分詞

#### 現在分詞の前置修飾・後置修飾 (No. 155)
```
doing + 名詞「〜している（名詞）」
名詞 + doing ...「〜している（名詞）」（他の語句を伴う場合は後置）
```

**例**: a **sleeping** baby（眠っている赤ちゃん）
**例**: the man **standing** over there（向こうに立っている男性）

#### 過去分詞句の後置修飾 (No. 157)
```
名詞 + done ...「〜された（名詞）」
```

**例**: the book **written** by him（彼によって書かれた本）

---

### 2. There be S doing/done (No. 160)

```
There is/are + S + doing「Sが〜している」
There is/are + S + done「Sが〜されている」
```

**例**: **There is** a cat **sleeping** on the sofa.（ソファで猫が眠っている）
**例**: **There are** many books **written** in English.

---

### 3. 補語となる分詞

#### 主格補語となる分詞（感情動詞）(No. 161)
```
S + V + doing/done（SVCの形で分詞が補語）
```

**例**: He sat **reading** a book.（彼は本を読みながら座っていた）
**例**: She looked **surprised**.（彼女は驚いているように見えた）

#### 目的格補語となる分詞 (No. 162)
```
S + V + O + doing「Oが〜しているのを...」
S + V + O + done「Oが〜されるのを...」
```

**例**: I saw him **crossing** the street.（彼が通りを渡っているのを見た）

---

### 4. make oneself understood/heard (No. 167)

```
make oneself understood「自分の意思を通じさせる」
make oneself heard「自分の声を届かせる」
```

**例**: I couldn't **make myself understood** in English.
（英語で自分の意思を伝えられなかった）

---

### 5. 分詞構文

#### 分詞を否定する語 (No. 170)
```
Not doing ...「〜しないで」
```

**例**: **Not knowing** what to do, I asked for help.
（何をすべきかわからなかったので、助けを求めた）

#### 完了分詞構文 (No. 171)
```
Having done ...「〜してから」（主節より前の時を表す）
```

**例**: **Having finished** my homework, I went out.
（宿題を終えてから、外出した）

#### 独立分詞構文 (No. 172)
```
主語 + doing ...「（主語が）〜して」（主節と主語が異なる）
```

**例**: **The sun having set**, we returned home.
（日が沈んだので、家に帰った）

#### 受動態の分詞構文 (No. 173)
```
(Being) done ...「〜されて」
```

**例**: **(Being) written** in easy English, the book is popular.
（やさしい英語で書かれているので、その本は人気がある）

---

### 6. 慣用的分詞構文 (No. 176)

| 表現 | 意味 |
|-----|-----|
| generally speaking | 一般的に言えば |
| frankly speaking | 率直に言えば |
| strictly speaking | 厳密に言えば |
| judging from A | Aから判断すると |
| speaking of A | Aと言えば |
| considering A | Aを考慮すると |

**例**: **Generally speaking**, Japanese people are polite.

---

### 7. with A done の付帯状況 (No. 177)

```
with + A + doing「Aが〜しながら」
with + A + done「Aが〜された状態で」
with + A + 形容詞「Aが...の状態で」
```

**例**: He sat **with his arms folded**.（腕を組んで座っていた）
**例**: She spoke **with her eyes closed**.（目を閉じて話した）

# 比較・特殊な前置詞 pair_001 - 習得すべき知識（整序英作文）

## 📚 このペアで学ぶ文法事項

### 1. 倍数表現 (No. 181)

```
... times as + 原級 + as A「Aの〜倍...」
```

**例**: This room is **three times as large as** that one.
（この部屋はあの部屋の3倍の広さだ）

**例**: He earns **twice as much as** I do.
（彼は私の2倍稼ぐ）

**バリエーション**:
- twice（2倍）
- three times（3倍）
- half（半分）

---

### 2. not so much A as B (No. 185)

```
not so much A as B「AというよりはむしろB」
= B rather than A
```

**例**: He is **not so much** a scholar **as** a journalist.
（彼は学者というよりむしろジャーナリストだ）

---

### 3. without so much as doing (No. 186)

```
without so much as doing「〜さえしないで」
```

**例**: He left **without so much as saying** goodbye.
（彼はさよならさえ言わずに去った）

---

### 4. 比較級の強調 (No. 188)

```
much / far / a lot / even / still + 比較級「ずっと〜」
```

**例**: This is **much better** than that.
（これはあれよりずっと良い）

**例**: She is **far more intelligent** than he is.
（彼女は彼よりずっと賢い）

⚠️ **very** は比較級を修飾**できない**。

---

### 5. 経過を表す in (No. 390)

```
in + 時間「〜後に」「〜たてば」
```

**例**: I'll be back **in** an hour.
（1時間後に戻ります）

**例**: He will finish the work **in** a week.
（彼は1週間で仕事を終えるだろう）

---

### 6. by「〜までに」(No. 393, 394)

```
by + 時間「〜までに」（期限）
until + 時間「〜まで（ずっと）」（継続）
```

**例**: Please finish the report **by** Friday.
（金曜日までにレポートを終えてください）

**例**: I'll wait **until** you come back.
（あなたが戻るまで待ちます）

---

## 🔑 整序英作文のコツ

### 語順の確認ポイント

1. **倍数 + as + 原級 + as** の語順を守る
2. **not so much A as B** の A と B の位置
3. **without so much as doing** は決まった形
4. **much/far + 比較級** の語順

### よくあるミス

| 誤り | 正しい形 |
|-----|---------|
| ✕ three times larger as | ⭕ three times as large as |
| ✕ very more | ⭕ much more |
| ✕ in three hours ago | ⭕ three hours ago / in three hours |

---

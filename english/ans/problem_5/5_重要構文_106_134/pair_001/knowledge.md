# 重要構文 pair_001 - 習得すべき知識（整序英作文）

## 📚 このペアで学ぶ文法事項

### 1. too ... to do (No. 120)

```
too + 形容詞/副詞 + to do「〜すぎて...できない」
```

**例**: He is **too young to** drive.
（彼は若すぎて運転できない）

**例**: This problem is **too difficult to** solve.
（この問題は難しすぎて解けない）

⚠️ 意味上の主語がある場合：
```
too + 形容詞 + for A + to do「Aにとって〜すぎて...できない」
```

---

### 2. 原形不定詞 (No. 126)

使役動詞・知覚動詞の後では原形不定詞を使う。

```
make/let/have + O + do（原形）
see/hear/feel + O + do（原形）
```

**例**: I saw him **cross** the street.
（彼が通りを渡るのを見た）

---

### 3. so ... that S + V (No. 353)

```
so + 形容詞/副詞 + that + S + V「とても〜なので...」
```

**例**: He is **so tall that** he can reach the ceiling.
（彼はとても背が高いので天井に手が届く）

**語順注意**:
```
so + 形容詞 + a/an + 名詞 + that ...
```

**例**: It was **so good a book that** I read it twice.

---

### 4. whether ... or not (No. 371)

```
whether ... or not「〜かどうか」
whether A or B「AかBか」
```

**例**: I don't know **whether** he will come **or not**.
（彼が来るかどうかわからない）

**例**: **Whether** you like it **or** not, you have to do it.
（好きかどうかに関わらず、やらなければならない）

---

### 5. not only A but (also) B (No. 376)

```
not only A but (also) B「AだけでなくBも」
= B as well as A
```

**例**: He is **not only** a teacher **but also** a writer.
（彼は教師であるだけでなく作家でもある）

⚠️ 動詞の数の一致は**B**に合わせる：
**例**: Not only he but also I **am** wrong.

---

### 6. have no choice but to do (No. 813)

```
have no choice but to do「〜せざるを得ない」
```

**例**: I **had no choice but to** accept his offer.
（彼の申し出を受け入れざるを得なかった）

---

## 🔑 整序英作文のコツ

### 構文の語順整理

| 構文 | 語順 |
|-----|-----|
| too ... to | too + 形容詞 + to do |
| so ... that | so + 形容詞 + that + S + V |
| not only ... but also | not only A but also B |

### よくあるミス

| 誤り | 正しい形 |
|-----|---------|
| ✕ too much difficult | ⭕ too difficult |
| ✕ so a good book that | ⭕ so good a book that |
| ✕ have no choice but do | ⭕ have no choice but **to** do |

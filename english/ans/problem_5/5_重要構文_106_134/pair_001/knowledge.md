# 重要構文 pair_001 - 習得すべき知識（整序英作文）

## 📚 このペアで学ぶ文法事項

### 1. too ... to do (No. 120)

```
too + 形容詞/副詞 + to do「〜すぎて...できない」
```

**例**: He is **too young to** drive.
（彼は若すぎて運転できない）

**例**: This problem is **too difficult to** solve.
（この問題は難しすぎて解けない）

> 🎵 **洋楽で覚える too ... to**
> Michael Jacksonの "You Are Not Alone" に "Too long ago, too hard to remember"（昔すぎて思い出せない）という歌詞があります。「〜すぎて...できない」という否定の意味を自然に理解できます。

⚠️ 意味上の主語がある場合：
```
too + 形容詞 + for A + to do「Aにとって〜すぎて...できない」
```

> 💡 **so ... that との書き換え**
> ```
> He is too young to drive.
> = He is so young that he can't drive.
> ```
> too ... to は「否定」が隠れていることを忘れずに！

### 2. 原形不定詞 (No. 126)

使役動詞・知覚動詞の後では原形不定詞を使う。

```
make/let/have + O + do（原形）
see/hear/feel + O + do（原形）
```

**例**: I saw him **cross** the street.
（彼が通りを渡るのを見た）

> 🎯 **なぜ原形なの？**
> 使役動詞・知覚動詞は「O が do するのを直接見る/させる」という意味。to不定詞の to は「〜に向かって」という距離感があるため、直接的な動作には使わないのです。

> 🔄 **受動態になると to が復活！**
> ```
> I saw him cross the street.（能動態：原形）
> → He was seen to cross the street.（受動態：to不定詞）
> ```
> 受動態では距離感が生まれるため、to が必要になります。

### 3. so ... that S + V (No. 353)

```
so + 形容詞/副詞 + that + S + V「とても〜なので...」
```

**例**: He is **so tall that** he can reach the ceiling.
（彼はとても背が高いので天井に手が届く）

**語順注意**:
```
so + 形容詞 + a/an + 名詞 + that ...
```

**例**: It was **so good a book that** I read it twice.

> ⚠️ **超頻出！語順の罠**
> 
> | 表現 | 語順 | 例 |
> |-----|-----|-----|
> | so | so + 形容詞 + a + 名詞 | so good a book |
> | such | such + a + 形容詞 + 名詞 | such a good book |
> | too | too + 形容詞 + a + 名詞 | too good a book |
> | as/how | as/how + 形容詞 + a + 名詞 | how good a book |
> 
> **覚え方**: 「so, too, as, how」は形容詞を先に引っ張る！

> 🔄 **so ... that = such ... that 書き換え**
> ```
> so good a book that = such a good book that
> ```
> 意味は同じ！でも語順が違う。整序問題では「so か such か」をまず確認！

### 4. whether ... or not (No. 371)

```
whether ... or not「〜かどうか」
whether A or B「AかBか」
```

**例**: I don't know **whether** he will come **or not**.
（彼が来るかどうかわからない）

**例**: **Whether** you like it **or** not, you have to do it.
（好きかどうかに関わらず、やらなければならない）

> 🔄 **whether vs if**
> 
> | 場面 | whether | if |
> |-----|---------|-----|
> | 主語として | ⭕ Whether he comes is unknown. | ✕ |
> | 前置詞の後 | ⭕ It depends on whether... | ✕ |
> | or not 直後 | ⭕ whether or not | ✕ if or not |
> | 文頭で | ⭕ Whether you like it or not, ... | ✕ |
> | それ以外 | ⭕ | ⭕ |
> 
> **結論**: 迷ったら whether を使えば安全！

---

### 5. not only A but (also) B (No. 376)

```
not only A but (also) B「AだけでなくBも」
= B as well as A
```

**例**: He is **not only** a teacher **but also** a writer.
（彼は教師であるだけでなく作家でもある）

⚠️ 動詞の数の一致は**B**に合わせる：
**例**: Not only he but also I **am** wrong.

> 💡 **B as well as A との書き換え**
> ```
> not only A but also B = B as well as A
> ```
> 「A だけでなく B も」= 「A と同様に B も」
> 
> **注意**: as well as A では動詞は**B**（主語）に一致！
> ```
> Not only he but also I am wrong.
> = I as well as he am wrong.
> ```

> 🎯 **強調の倒置**
> not only が文頭に来ると倒置が起きる：
> ```
> Not only is he a teacher, but he is also a writer.
> ```
> 否定語（句）が文頭 → 疑問文の語順になる！

---

### 6. have no choice but to do (No. 813)

```
have no choice but to do「〜せざるを得ない」
```

**例**: I **had no choice but to** accept his offer.
（彼の申し出を受け入れざるを得なかった）

> 📖 **語源で理解！**
> - but = except（〜以外）
> - 直訳:「〜すること以外に選択肢がない」
> → 「〜せざるを得ない」
> 
> 同じ構造: cannot but do / cannot help doing（〜せずにはいられない）

> 🔄 **言い換えパターン**
> ```
> have no choice but to do
> = cannot but do（やや古風）
> = cannot help doing
> = have no option but to do
> ```

## 🔑 整序英作文のコツ

### 構文の語順整理

| 構文 | 語順 | ポイント |
|-----|-----|---------|
| too ... to | too + 形容詞 + to do | 否定の意味が隠れている |
| so ... that | so + 形容詞 + that + S + V | 形容詞が先！ |
| such ... that | such + a + 形容詞 + 名詞 + that | a が先！ |
| not only ... but also | not only A but also B | 動詞はBに一致 |
| have no choice but to | have no choice but to do | but = except |

### よくあるミス

| 誤り | 正しい形 | なぜ？ |
|-----|---------|-------|
| ✕ too much difficult | ⭕ too difficult | much は不要 |
| ✕ so a good book that | ⭕ so good a book that | 形容詞が先 |
| ✕ have no choice but do | ⭕ have no choice but **to** do | to を忘れない |
| ✕ if or not | ⭕ whether or not | if は or not 直後不可 |

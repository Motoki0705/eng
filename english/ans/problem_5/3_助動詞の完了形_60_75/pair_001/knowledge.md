# 助動詞の完了形 pair_001 - 習得すべき知識（整序英作文）

## 📚 このペアで学ぶ文法事項

### 1. 過去への推量

#### must have done (No. 60)
```
must have done「〜したに違いない」（過去の確信）
```

**例**: He **must have been** tired.
（彼は疲れていたに違いない）

> 🔍 **探偵の推理のような確信**
> シャーロック・ホームズが "He must have entered through the window."（彼は窓から入ったに違いない）と言うイメージ。証拠から過去を推測する「確信」を表します。

#### can't have done (No. 61)
```
can't have done「〜したはずがない」（過去の否定的確信）
```

**例**: She **can't have said** such a thing.
（彼女がそんなことを言ったはずがない）

> 💡 **can't = couldn't でもOK**
> "She couldn't have said that." も同じ意味。couldn't の方がやや丁寧・フォーマルな響きがあります。

#### may have done (No. 62)
```
may have done「〜したかもしれない」（過去の推量）
```

**例**: He **may have missed** the train.
（彼は電車に乗り遅れたかもしれない）

> 🎯 **確信度を比較！**
> ```
> 確信度 高 ←――――――――――――――――→ 低
> must have   can't have   may/might have
>  (90%)        (90%)         (50%)
> 違いない    はずがない    かもしれない
> ```

### 2. 過去への後悔・非難

#### should have done (No. 64)
```
should have done「〜すべきだった（のにしなかった）」
```

**例**: You **should have told** me the truth.
（本当のことを言うべきだった）

> 😢 **後悔の定番フレーズ！**
> "I should have studied harder."（もっと勉強すべきだった）
> これは試験後の後悔でよく使う表現。映画やドラマでも頻出！

> 🎬 **映画での使用例**
> 『タイタニック』でローズが "I should have let go."（手を離すべきだった）というセリフがあります。この構文は「後悔」を表す最も一般的な表現です。

#### should not have done (No. 66)
```
should not have done「〜すべきではなかった（のにした）」
```

**例**: You **should not have bought** such an expensive thing.
（そんな高いものを買うべきではなかった）

> 🔄 **should have vs shouldn't have**
> - should have done → やらなかったことへの後悔
> - shouldn't have done → やったことへの後悔
> 
> どちらも「実際は逆だった」という含意があります。

#### ought not to have done (No. 67)
```
ought not to have done「〜すべきではなかった」
```

**例**: He **ought not to have said** that.
（彼はそんなことを言うべきではなかった）

> 🇬🇧 **ought to はイギリス英語で多用**
> アメリカ英語では should を使うことが多いですが、イギリスでは ought to が日常的に使われます。Harry Potter シリーズでも "You ought to..." がよく出てきます。

---

### 3. needn't have done (No. 68)

```
needn't have done「〜する必要はなかった（のにした）」
```

**例**: You **needn't have hurried**.
（急ぐ必要はなかったのに）

⚠️ didn't need to do との違い:
- needn't have done: 実際にやった
- didn't need to do: やったかどうか不明

> 🎭 **2つの「必要なかった」の決定的な違い**
> 
> | 表現 | 実際の行動 | 例 |
> |-----|----------|---|
> | needn't have done | やった😓 | You needn't have bought it.（買う必要なかったのに買った）|
> | didn't need to do | 不明/やらなかった | I didn't need to buy it.（買う必要がなかった）|
> 
> needn't have done には「無駄だったね」「徒労だったね」というニュアンスがあります。

## 🔑 整序英作文のコツ

### 助動詞 + have + 過去分詞 の意味整理

| 表現 | 意味 | ニュアンス | 確信度 |
|-----|-----|----------|-------|
| must have done | 〜したに違いない | 確信（肯定） | 90% |
| can't have done | 〜したはずがない | 確信（否定） | 90% |
| may have done | 〜したかもしれない | 推量 | 50% |
| might have done | 〜したかもしれない | やや控えめな推量 | 30% |
| should have done | 〜すべきだった | 後悔・非難 | ー |
| should not have done | 〜すべきでなかった | 後悔・非難 | ー |
| ought to have done | 〜すべきだった | should より堅い | ー |
| needn't have done | 〜する必要はなかった | 実際にやった | ー |

> 💡 **覚え方のコツ**
> 「助動詞 + have + p.p.」を見たら、まず「過去のことを言っている」と認識！
> - 推量系（must/may/might/can't）→ 過去への推測
> - should/ought to → 過去への後悔
> - needn't → 無駄だった

### 語順のポイント

```
助動詞 + have + 過去分詞
```

この3語がセットになることを意識する。

> ⚠️ **よくある間違い**
> ✕ must had done → ○ must have done
> ✕ should has done → ○ should have done
> 
> 助動詞の後は必ず「原形」！ have は had や has にならない！

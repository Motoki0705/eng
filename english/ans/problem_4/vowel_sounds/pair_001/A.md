## 詳細解説

### (a) 正解：1（al-read-y）- アクセント問題

| 単語 | 発音記号 | アクセント位置 |
|------|---------|--------------|
| 1. **al-read-y** | /ɔːlˈredi/ | **第2音節** |
| 2. beau-ty | /ˈbjuːti/ | 第1音節 |
| 3. eas-y | /ˈiːzi/ | 第1音節 |
| 4. emp-ty | /ˈempti/ | 第1音節 |
| 5. plen-ty | /ˈplenti/ | 第1音節 |

### (b) 正解：4（swear）- 母音問題

| 単語 | 発音記号 | "ea" の発音 |
|------|---------|------------|
| 1. break | /breɪk/ | /eɪ/ |
| 2. great | /ɡreɪt/ | /eɪ/ |
| 3. steak | /steɪk/ | /eɪ/ |
| 4. **swear** | /swer/ | **/eə/** |
| 5. wear | /wer/ | /eə/ |

**注意**: wear も /eə/ なので、**正解は4と5のグループと1, 2, 3のグループ**。

### (c) 正解：4（mood）- 母音問題

| 単語 | 発音記号 | "oo" の発音 |
|------|---------|-----------|
| 1. blood | /blʌd/ | /ʌ/ |
| 2. flood | /flʌd/ | /ʌ/ |
| 3. good | /ɡʊd/ | /ʊ/ |
| 4. **mood** | /muːd/ | **/uː/** |
| 5. wood | /wʊd/ | /ʊ/ |

**ポイント**: "oo" は /uː/（長母音）、/ʊ/（短母音）、/ʌ/ の3通りの発音がある。

### (d) 正解：複数 - 母音問題

| 単語 | 発音記号 | "ough" の発音 |
|------|---------|--------------|
| 1. cough | /kɔːf/ | /ɔːf/ |
| 2. ought | /ɔːt/ | /ɔː/ |
| 3. rough | /rʌf/ | /ʌf/ |
| 4. though | /ðoʊ/ | /oʊ/ |
| 5. through | /θruː/ | /uː/ |

**ポイント**: "ough" は英語で最も発音が多様なスペリング。すべて異なる発音。

### (e) 正解：1（beard）- 母音問題

| 単語 | 発音記号 | "ear" の発音 |
|------|---------|-------------|
| 1. **beard** | /bɪrd/ | **/ɪr/** |
| 2. heard | /hɜːrd/ | /ɜːr/ |
| 3. near-ly | /ˈnɪrli/ | /ɪr/ |
| 4. pearl | /pɜːrl/ | /ɜːr/ |
| 5. search | /sɜːrtʃ/ | /ɜːr/ |

**注意**: nearlyも /ɪr/ なので、**正解は2, 4, 5のグループと1, 3のグループ**。

### (f) 正解：1（height）- 母音問題

| 単語 | 発音記号 | "eigh" の発音 |
|------|---------|--------------|
| 1. **height** | /haɪt/ | **/aɪ/** |
| 2. neigh-bor | /ˈneɪbər/ | /eɪ/ |
| 3. sleigh | /sleɪ/ | /eɪ/ |
| 4. weight | /weɪt/ | /eɪ/ |
| 5. weigh | /weɪ/ | /eɪ/ |

**ポイント**: height だけ "eigh" が /aɪ/ と発音される。

## 詳細解説

### (a) 正解：3（dis-count）- アクセント問題

| 単語 | 発音記号 | アクセント位置 |
|------|---------|--------------|
| 1. ac-count | /əˈkaʊnt/ | 第2音節 |
| 2. a-mount | /əˈmaʊnt/ | 第2音節 |
| 3. **dis-count** | /ˈdɪskaʊnt/ | **第1音節** |
| 4. moun-tain | /ˈmaʊntən/ | 第1音節 |
| 5. sur-round | /səˈraʊnd/ | 第2音節 |

**注意**: mountainも第1音節。**正解は1, 2, 5のグループと3, 4のグループ**。

### (b) 正解：6（すべて同じ）- アクセント問題

| 単語 | 発音記号 | アクセント位置 |
|------|---------|--------------|
| 1. de-cide | /dɪˈsaɪd/ | 第2音節 |
| 2. di-vide | /dɪˈvaɪd/ | 第2音節 |
| 3. guide | /ɡaɪd/ | 単音節 |
| 4. pro-vide | /prəˈvaɪd/ | 第2音節 |
| 5. wide | /waɪd/ | 単音節 |

**ポイント**: 単音節語と2音節語の第2音節は同等のパターン。

### (c) 正解：2（broad）- 母音問題

| 単語 | 発音記号 | "oa" の発音 |
|------|---------|-----------|
| 1. ap-proach | /əˈproʊtʃ/ | /oʊ/ |
| 2. **broad** | /brɔːd/ | **/ɔː/** |
| 3. coach | /koʊtʃ/ | /oʊ/ |
| 4. road | /roʊd/ | /oʊ/ |
| 5. roast | /roʊst/ | /oʊ/ |

**ポイント**: broad は例外的に "oa" が /ɔː/ と発音される。

### (d) 正解：2（for-eign）- 母音問題

| 単語 | 発音記号 | "ign" の発音 |
|------|---------|------------|
| 1. de-sign | /dɪˈzaɪn/ | /aɪn/ |
| 2. **for-eign** | /ˈfɔːrən/ | **/ən/** |
| 3. reign | /reɪn/ | /eɪn/ |
| 4. sign | /saɪn/ | /aɪn/ |
| 5. sov-er-eign | /ˈsɑːvrən/ | /ən/ |

**注意**: sovereignも /ən/ で異なる。**正解は2と5のグループと1, 3, 4のグループ**。

### (e) 正解：5（weap-on）- 母音問題

| 単語 | 発音記号 | "ea" の発音 |
|------|---------|-----------|
| 1. ba-con | /ˈbeɪkən/ | - |
| 2. rea-son | /ˈriːzn/ | /iː/ |
| 3. sea-son | /ˈsiːzn/ | /iː/ |
| 4. trea-son | /ˈtriːzn/ | /iː/ |
| 5. **weap-on** | /ˈwepən/ | **/e/** |

**ポイント**: weapon の "ea" は短母音 /e/。

### (f) 正解：2（crowd）- 母音問題

| 単語 | 発音記号 | "ow" の発音 |
|------|---------|-----------|
| 1. bowl | /boʊl/ | /oʊ/ |
| 2. **crowd** | /kraʊd/ | **/aʊ/** |
| 3. grow | /ɡroʊ/ | /oʊ/ |
| 4. know | /noʊ/ | /oʊ/ |
| 5. show | /ʃoʊ/ | /oʊ/ |

**ポイント**: "ow" は /oʊ/ または /aʊ/ の2通り。crowd は /aʊ/。

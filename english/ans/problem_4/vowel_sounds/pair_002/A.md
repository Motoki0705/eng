## vowel_sounds pair_002 解答A

---

## 正解

| 問題 | 正解 | 解説 |
|:---:|:---:|------|
| (a) | **1** | bury の "u" は /e/、他は /ʌ/ |
| (b) | **1** | brow の "ow" は /aʊ/、他は /oʊ/ |
| (c) | **3** | earth の "ea" は /ɜː/、他は /e/ |
| (d) | **6** | すべて黙字を含む（発音されない子音） |
| (e) | **5** | tomb の "b" は発音される |
| (f) | **4** | science は2音節、他は -cient/-tient |

---

## 詳細解説

### (a) 正解：1（bur-y）- 母音問題

| 単語 | 発音記号 | "u" の発音 |
|------|---------|----------|
| 1. **bur-y** | /ˈberi/ | **/e/** |
| 2. cur-ry | /ˈkɜːri/ | /ɜː/ |
| 3. fur-ry | /ˈfɜːri/ | /ɜː/ |
| 4. hur-ry | /ˈhɜːri/ | /ɜː/ |
| 5. wor-ry | /ˈwɜːri/ | /ɜː/ |

**ポイント**: bury は例外的に /e/ と発音される（berry と同音）。

### (b) 正解：1（brow）- 母音問題

| 単語 | 発音記号 | "ow" の発音 |
|------|---------|-----------|
| 1. **brow** | /braʊ/ | **/aʊ/** |
| 2. flow | /floʊ/ | /oʊ/ |
| 3. grow | /ɡroʊ/ | /oʊ/ |
| 4. low | /loʊ/ | /oʊ/ |
| 5. row | /roʊ/ | /oʊ/ |

**ポイント**: "ow" は /oʊ/ か /aʊ/ のどちらか。brow は /aʊ/。

### (c) 正解：3（earth）- 母音問題

| 単語 | 発音記号 | "ea" の発音 |
|------|---------|-----------|
| 1. breath | /breθ/ | /e/ |
| 2. death | /deθ/ | /e/ |
| 3. **earth** | /ɜːrθ/ | **/ɜː/** |
| 4. health | /helθ/ | /e/ |
| 5. wealth | /welθ/ | /e/ |

**ポイント**: earth の "ear" は /ɜː/、他の "ea" は /e/。

### (d) 正解：6（すべて同じ）- 黙字問題

| 単語 | 発音記号 | 黙字 |
|------|---------|-----|
| 1. is-land | /ˈaɪlənd/ | "s" が黙字 |
| 2. lis-ten | /ˈlɪsn/ | "t" が黙字 |
| 3. mus-cle | /ˈmʌsl/ | "c" が黙字 |
| 4. whis-tle | /ˈwɪsl/ | "t" が黙字 |
| 5. wres-tle | /ˈresl/ | "t" が黙字 |

**ポイント**: すべて黙字（発音されない子音）を含む。

### (e) 正解：5（tomb）- 黙字問題

| 単語 | 発音記号 | "b" の発音 |
|------|---------|----------|
| 1. bomb | /bɑːm/ | 黙字 |
| 2. climb | /klaɪm/ | 黙字 |
| 3. comb | /koʊm/ | 黙字 |
| 4. thumb | /θʌm/ | 黙字 |
| 5. **tomb** | /tuːm/ | 黙字 |

**ポイント**: すべて語末の "mb" の "b" は黙字。**正解は6**。

### (f) 正解：4（sci-ence）- アクセント問題

| 単語 | 発音記号 | アクセント位置 |
|------|---------|--------------|
| 1. an-cient | /ˈeɪnʃənt/ | 第1音節 |
| 2. ef-fi-cient | /ɪˈfɪʃnt/ | 第2音節 |
| 3. pa-tient | /ˈpeɪʃnt/ | 第1音節 |
| 4. **sci-ence** | /ˈsaɪəns/ | **第1音節** |
| 5. suf-fi-cient | /səˈfɪʃnt/ | 第2音節 |

**注意**: ancientとpatientは第1音節、efficientとsufficientは第2音節。**正解は複数のグループに分かれる**。

---

## 発音規則まとめ

| パターン | 例 | 説明 |
|---------|---|------|
| bury の例外 | bury = /ˈberi/ | "u" が /e/ と発音 |
| ow の二重発音 | flow /oʊ/ vs brow /aʊ/ | 語によって異なる |
| mb の黙字 | bomb, climb, comb | 語末の "b" は発音しない |
| ear の発音 | earth /ɜː/ vs hear /ɪr/ | 続く子音で変わる |

# 🔊 子音とアクセント：音の「壁」を理解する

## 🎵 子音の役割

子音は母音の**間に立つ壁**のような存在です。英語のリズムを理解するには、子音の働きを知ることが重要です。

### 有声音と無声音

| 無声音 | 有声音 | 違い |
|--------|-------|------|
| /p/ | /b/ | 声帯の振動 |
| /t/ | /d/ | 声帯の振動 |
| /k/ | /g/ | 声帯の振動 |
| /f/ | /v/ | 声帯の振動 |
| /s/ | /z/ | 声帯の振動 |
| /θ/ (th) | /ð/ (th) | 声帯の振動 |
| /ʃ/ (sh) | /ʒ/ (zh) | 声帯の振動 |
| /tʃ/ (ch) | /dʒ/ (j) | 声帯の振動 |

---

## 📐 子音と音節の境界

### 音節の分け方

子音は**音節の境界**を決定します：

| 単語 | 音節分け | 規則 |
|------|---------|------|
| water | wa-ter | VCV → V-CV |
| happy | hap-py | VCCV → VC-CV |
| camera | cam-e-ra | VCV → V-CV |

VCV（母音-子音-母音）パターンでは、子音は**後ろの音節**に属することが多いです。

---

## 🔍 子音連結（クラスター）

### 2つ以上の子音が連続

英語では、複数の子音が連続することがあります：

| 位置 | 例 | クラスター |
|------|-----|----------|
| 語頭 | **str**eet | /str/ |
| 語頭 | **spl**it | /spl/ |
| 語末 | te**xts** | /ksts/ |
| 語末 | stren**gths** | /ŋθs/ |

日本語にはない組み合わせなので、発音が難しいです。

## 📊 発音されない子音（黙字）

### 綴りにあるが発音されない

| 黙字 | 例 |
|------|-----|
| k | **k**now, **k**nife, **k**nee |
| g | **g**nat, si**g**n, forei**g**n |
| b | com**b**, clim**b**, dum**b** |
| w | **w**rite, **w**rong, ans**w**er |
| h | **h**our, **h**onest, **h**eir |
| l | wa**l**k, ta**l**k, ha**l**f |
| t | lis**t**en, fas**t**en, cas**t**le |

---

## 🎯 子音の同化

### 隣の音に影響される

連続した子音は、互いに影響を与えます：

| 元の音 | 同化後 | 例 |
|--------|-------|-----|
| /t/ + /j/ | /tʃ/ | don't you → /doʊntʃə/ |
| /d/ + /j/ | /dʒ/ | did you → /dɪdʒə/ |
| /s/ + /j/ | /ʃ/ | miss you → /mɪʃə/ |
| /z/ + /j/ | /ʒ/ | as you → /æʒə/ |

自然な会話では、これらの同化が起きます。

---

## 📐 語末の -ed の発音

### 3種類の発音

| 規則 | 発音 | 例 |
|------|------|-----|
| /t/, /d/ の後 | /ɪd/ | wanted, needed |
| 無声音の後 | /t/ | walked, stopped |
| 有声音の後 | /d/ | played, called |

これは**音節数**にも影響します：
- walked → 1音節（/wɔːkt/）
- wanted → 2音節（/wɒn.tɪd/）

---

## 💡 この単元で習得する感覚

| 概念 | ポイント |
|------|---------|
| 有声/無声 | 声帯の振動の有無 |
| 子音連結 | 日本語にない組み合わせ |
| 黙字 | 綴りにあるが発音しない |
| -ed の発音 | 前の音で決まる |

---

## 📚 豆知識：なぜ黙字があるのか？

黙字の多くは、**昔は発音されていた**音です：

| 単語 | 中世英語の発音 | 現代の発音 |
|------|--------------|----------|
| knight | /kniçt/ | /naɪt/ |
| know | /knɔːw/ | /noʊ/ |
| write | /ˈwriːtə/ | /raɪt/ |

印刷技術の発達で綴りが固定された後、発音だけが変化したため、「発音されない文字」が残りました。

## ⚠️ 試験での注意

### 発音問題での子音

アクセント問題と異なり、**発音問題**では子音の発音を問われることがあります：

| パターン | 例 |
|---------|-----|
| th の発音 | /θ/ vs /ð/ |
| s の発音 | /s/ vs /z/ |
| -ed の発音 | /t/ vs /d/ vs /ɪd/ |

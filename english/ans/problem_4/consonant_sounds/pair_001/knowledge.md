# 🔊 子音とアクセント：音の「壁」を理解する

## 🎵 子音の役割

子音は母音の**間に立つ壁**のような存在です。英語のリズムを理解するには、子音の働きを知ることが重要です。

### 有声音と無声音

| 無声音 | 有声音 | 違い |
|--------|-------|------|
| /p/ | /b/ | 声帯の振動 |
| /t/ | /d/ | 声帯の振動 |
| /k/ | /g/ | 声帯の振動 |
| /f/ | /v/ | 声帯の振動 |
| /s/ | /z/ | 声帯の振動 |
| /θ/ (th) | /ð/ (th) | 声帯の振動 |
| /ʃ/ (sh) | /ʒ/ (zh) | 声帯の振動 |
| /tʃ/ (ch) | /dʒ/ (j) | 声帯の振動 |

### 🎭 「有声」と「無声」を体感する方法

のどに手を当てて「zzzzz」と言ってみてください。振動を感じますか？

次に「sssss」と言ってみましょう。振動が止まったはずです。

これが**有声音と無声音の違い**。口の形は同じなのに、声帯を使うか使わないかだけが違うのです。

この区別は日本語にもありますが（「か」vs「が」）、英語では**語末**でも区別が重要です。`bus` /bʌs/ と `buzz` /bʌz/ は全く違う単語！

## 📐 子音と音節の境界

### 音節の分け方

子音は**音節の境界**を決定します：

| 単語 | 音節分け | 規則 |
|------|---------|------|
| water | wa-ter | VCV → V-CV |
| happy | hap-py | VCCV → VC-CV |
| camera | cam-e-ra | VCV → V-CV |

VCV（母音-子音-母音）パターンでは、子音は**後ろの音節**に属することが多いです。

---

## 🔍 子音連結（クラスター）

### 2つ以上の子音が連続

英語では、複数の子音が連続することがあります：

| 位置 | 例 | クラスター |
|------|-----|----------|
| 語頭 | **str**eet | /str/ |
| 語頭 | **spl**it | /spl/ |
| 語末 | te**xts** | /ksts/ |
| 語末 | stren**gths** | /ŋθs/ |

日本語にはない組み合わせなので、発音が難しいです。

### 🌍 子音クラスターと言語の「性格」

言語学者は、子音クラスターの複雑さで言語を分類します：

| 言語 | クラスターの複雑さ | 例 |
|------|-----------------|-----|
| 日本語 | とても単純 | 子音は基本1つ |
| スペイン語 | 単純 | tr, pl くらい |
| 英語 | 中程度 | str, spl, nkths |
| ロシア語 | 複雑 | 5つ以上連続可能 |
| グルジア語 | 超複雑 | 8つ連続の例あり |

英語の子音クラスターは「中程度」ですが、日本語話者にとっては十分難しい。

**コツ**：クラスターを分解せず、**一息で**発音する練習をしましょう。「ストリート」ではなく「stri:t」と一塊で。

## 📊 発音されない子音（黙字）

### 綴りにあるが発音されない

| 黙字 | 例 |
|------|-----|
| k | **k**now, **k**nife, **k**nee |
| g | **g**nat, si**g**n, forei**g**n |
| b | com**b**, clim**b**, dum**b** |
| w | **w**rite, **w**rong, ans**w**er |
| h | **h**our, **h**onest, **h**eir |
| l | wa**l**k, ta**l**k, ha**l**f |
| t | lis**t**en, fas**t**en, cas**t**le |

## 🎯 子音の同化

### 隣の音に影響される

連続した子音は、互いに影響を与えます：

| 元の音 | 同化後 | 例 |
|--------|-------|-----|
| /t/ + /j/ | /tʃ/ | don't you → /doʊntʃə/ |
| /d/ + /j/ | /dʒ/ | did you → /dɪdʒə/ |
| /s/ + /j/ | /ʃ/ | miss you → /mɪʃə/ |
| /z/ + /j/ | /ʒ/ | as you → /æʒə/ |

自然な会話では、これらの同化が起きます。

## 📐 語末の -ed の発音

### 3種類の発音

| 規則 | 発音 | 例 |
|------|------|-----|
| /t/, /d/ の後 | /ɪd/ | wanted, needed |
| 無声音の後 | /t/ | walked, stopped |
| 有声音の後 | /d/ | played, called |

これは**音節数**にも影響します：
- walked → 1音節（/wɔːkt/）
- wanted → 2音節（/wɒn.tɪd/）

### 🎯 なぜ -ed は3種類の発音があるのか？

これは**発音のしやすさ**（調音の容易さ）によります。

/t/ や /d/ の後にまた /t/ や /d/ を発音するのは難しい（try saying "want-d"）。だから母音 /ɪ/ を挿入して発音しやすくする。

無声音の後に有声音の /d/ を発音するのも不自然。だから無声化して /t/ になる。

これは**同化**（assimilation）と呼ばれる現象で、人間の口が「楽をしたがる」証拠です。

入試では -ed の発音を問う問題が出ます。「音節が増えるかどうか」がポイントです。

| 子音連結 | 日本語にない組み合わせ |

## 📚 豆知識：なぜ黙字があるのか？

黙字の多くは、**昔は発音されていた**音です：

| 単語 | 中世英語の発音 | 現代の発音 |
|------|--------------|----------|
| knight | /kniçt/ | /naɪt/ |
| know | /knɔːw/ | /noʊ/ |
| write | /ˈwriːtə/ | /raɪt/ |

印刷技術の発達で綴りが固定された後、発音だけが変化したため、「発音されない文字」が残りました。

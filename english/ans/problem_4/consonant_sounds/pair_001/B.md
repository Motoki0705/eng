## consonant_sounds pair_001 解答B

---

## 正解

| 問題 | 正解 | 解説 |
|:---:|:---:|------|
| (a) | **5** | signal の "g" は発音される |
| (b) | **5** | channel の "ch" は /tʃ/、他は /ʃ/ |
| (c) | **6** | すべて "kn" は "n" のみ発音 |
| (d) | **6** | すべて "l" が黙字または /l/ |
| (e) | **2** | spaghetti の "gh" は /g/、他は様々 |
| (f) | **5** | worry の "wr" は w が発音される |

---

## 詳細解説

### (a) 正解：5（sig-nal）- "gn" の発音

| 単語 | 発音記号 | "g" の発音 |
|------|---------|----------|
| 1. gnat | /næt/ | 黙字 |
| 2. gnaw | /nɔː/ | 黙字 |
| 3. reign | /reɪn/ | 黙字 |
| 4. sign | /saɪn/ | 黙字 |
| 5. **sig-nal** | /ˈsɪgnəl/ | **/g/ 発音** |

**ポイント**: "gn" が語頭・語末にある場合 "g" は黙字。しかし signal のように音節境界を跨ぐ場合は発音される。

### (b) 正解：5（chan-nel）- "ch" の発音

| 単語 | 発音記号 | "ch" の発音 |
|------|---------|----------|
| 1. chef | /ʃef/ | /ʃ/ |
| 2. ma-chine | /məˈʃiːn/ | /ʃ/ |
| 3. bro-chure | /broʊˈʃʊr/ | /ʃ/ |
| 4. cham-pagne | /ʃæmˈpeɪn/ | /ʃ/ |
| 5. **chan-nel** | /ˈtʃænl/ | **/tʃ/** |

**ポイント**: フランス語由来の語では "ch" は /ʃ/。channel は英語本来の発音 /tʃ/。

### (c) 正解：6（すべて同じ）- "kn" = /n/

| 単語 | 発音記号 | "kn" の発音 |
|------|---------|----------|
| 1. knight | /naɪt/ | /n/ |
| 2. knife | /naɪf/ | /n/ |
| 3. knock | /nɑːk/ | /n/ |
| 4. know | /noʊ/ | /n/ |
| 5. knit | /nɪt/ | /n/ |

**ポイント**: 語頭の "kn" は常に /n/ のみ発音（"k" は黙字）。

### (d) 正解：6（すべて同じ）- 黙字 "l"

| 単語 | 発音記号 | "l" の発音 |
|------|---------|----------|
| 1. half | /hæf/ | 黙字 |
| 2. calf | /kæf/ | 黙字 |
| 3. chalk | /tʃɔːk/ | 黙字 |
| 4. folk | /foʊk/ | 黙字 |
| 5. folk-lore | /ˈfoʊklɔːr/ | 最初の l は黙字 |

**ポイント**: "-alf", "-alk", "-olk" のパターンでは "l" は黙字。

### (e) 正解：2（spa-ghet-ti）- "gh" の発音

| 単語 | 発音記号 | "gh" の発音 |
|------|---------|----------|
| 1. ghost | /goʊst/ | /g/ (goと同化) |
| 2. **spa-ghet-ti** | /spəˈgeti/ | **黙字** |
| 3. cough | /kɔːf/ | /f/ |
| 4. rough | /rʌf/ | /f/ |
| 5. though | /ðoʊ/ | 黙字 |

**ポイント**: "gh" は /f/, /g/, 黙字など様々な発音がある。

### (f) 正解：5（wor-ry）- "wr" の発音

| 単語 | 発音記号 | "w" の発音 |
|------|---------|----------|
| 1. write | /raɪt/ | 黙字 |
| 2. wrong | /rɔːŋ/ | 黙字 |
| 3. wrist | /rɪst/ | 黙字 |
| 4. wrap | /ræp/ | 黙字 |
| 5. **wor-ry** | /ˈwʌri/ | **/w/ 発音** |

**ポイント**: "wr-" の "w" は黙字だが、worry は "wr" ではなく "w + or" なので "w" が発音される。

## 子音発音規則まとめ（補足）

| パターン | 発音 | 例 |
|---------|-----|---|
| gn- (語頭) | /n/ | gnat, gnaw |
| -gn (語末) | /n/ | sign, reign |
| -gn- (音節内) | /gn/ | signal, signature |
| kn- | /n/ | know, knife |
| wr- | /r/ | write, wrong |
| ch (仏語由来) | /ʃ/ | chef, machine |
| ch (英語本来) | /tʃ/ | channel, church |

---

## "gh" の発音パターン

| 発音 | 条件 | 例 |
|-----|-----|---|
| /g/ | 語頭の ghost など | ghost, ghastly |
| /f/ | -ough, -augh | cough, laugh |
| 黙字 | -ight, -ough | night, though |
| 黙字 | イタリア語由来 | spaghetti |

## 外来語と発音

| 起源 | 特徴 | 例 |
|-----|-----|---|
| フランス語 | ch = /ʃ/ | chef, machine, champagne |
| イタリア語 | gh = 黙字 | spaghetti, ghetto |
| ギリシャ語 | ps, pn = s, n | psychology, pneumonia |
| ドイツ語 | ch = /k/ | ache, stomach |

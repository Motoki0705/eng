## 詳細解説

### (a) 正解：6（すべて同じ）- 黙字 "t"

| 単語 | 発音記号 | "t" の発音 |
|------|---------|----------|
| 1. cas-tle | /ˈkæsl/ | 黙字 |
| 2. fas-ten | /ˈfæsn/ | 黙字 |
| 3. has-ten | /ˈheɪsn/ | 黙字 |
| 4. lis-ten | /ˈlɪsn/ | 黙字 |
| 5. mois-ten | /ˈmɔɪsn/ | 黙字 |

**ポイント**: "-sten" や "-stle" のパターンでは "t" は発音されない。

### (b) 正解：6（すべて同じ）- "dge" = /dʒ/

| 単語 | 発音記号 | "dge" の発音 |
|------|---------|------------|
| 1. bridge | /brɪdʒ/ | /dʒ/ |
| 2. edge | /edʒ/ | /dʒ/ |
| 3. judge | /dʒʌdʒ/ | /dʒ/ |
| 4. lodge | /lɑːdʒ/ | /dʒ/ |
| 5. ridge | /rɪdʒ/ | /dʒ/ |

**ポイント**: "dge" は常に /dʒ/ と発音される。

### (c) 正解：4（sal-mon）- 黙字 "l"

| 単語 | 発音記号 | "l" の発音 |
|------|---------|----------|
| 1. calm | /kɑːm/ | 黙字 |
| 2. palm | /pɑːm/ | 黙字 |
| 3. psalm | /sɑːm/ | 黙字 |
| 4. **sal-mon** | /ˈsæmən/ | **黙字** |
| 5. qualm | /kwɑːm/ | 黙字 |

**ポイント**: すべて "l" は黙字。**正解は6**。

### (d) 正解：2（knowl-edge）- アクセント問題

| 単語 | 発音記号 | アクセント位置 |
|------|---------|--------------|
| 1. ac-knowl-edge | /əkˈnɑːlɪdʒ/ | 第2音節 |
| 2. **knowl-edge** | /ˈnɑːlɪdʒ/ | **第1音節** |
| 3. ledge | /ledʒ/ | 単音節 |
| 4. pledge | /pledʒ/ | 単音節 |
| 5. wedge | /wedʒ/ | 単音節 |

**ポイント**: acknowledgeは第2音節、knowledgeは第1音節。

### (e) 正解：4（sub-urb）- 黙字 "b"

| 単語 | 発音記号 | "b" の発音 |
|------|---------|----------|
| 1. debt | /det/ | 黙字 |
| 2. doubt | /daʊt/ | 黙字 |
| 3. sub-tle | /ˈsʌtl/ | 黙字 |
| 4. **sub-urb** | /ˈsʌbɜːrb/ | **発音される** |
| 5. thumb | /θʌm/ | 黙字 |

**ポイント**: suburb の "b" は発音される。debt, doubt, subtle, thumb の "b" は黙字。

### (f) 正解：6（すべて同じ）- "s" = /ʒ/

| 単語 | 発音記号 | "s" の発音 |
|------|---------|----------|
| 1. cas-u-al | /ˈkæʒuəl/ | /ʒ/ |
| 2. meas-ure | /ˈmeʒər/ | /ʒ/ |
| 3. pleas-ure | /ˈpleʒər/ | /ʒ/ |
| 4. treas-ure | /ˈtreʒər/ | /ʒ/ |
| 5. u-su-al | /ˈjuːʒuəl/ | /ʒ/ |

**ポイント**: 母音に挟まれた "s" は /ʒ/ と発音されることがある。

## 子音発音規則まとめ

| パターン | 例 | 説明 |
|---------|---|------|
| -sten, -stle | listen, castle | "t" は黙字 |
| -dge | bridge, judge | /dʒ/ と発音 |
| -lm, -lk | calm, talk | "l" は黙字（多くの場合） |
| -bt, -mb | debt, thumb | "b" は黙字 |
| -s- (母音間) | measure, usual | /ʒ/ と発音 |

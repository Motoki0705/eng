## accent_rules pair_002 解答B

---

## 正解

| 問題 | 正解 | アクセント規則 |
|:---:|:---:|--------------|
| (a) | **3** | husband は第1音節、他は第2音節 |
| (b) | **6** | すべて第1音節にアクセント |
| (c) | **1** | atmosphere は第1音節、他は第2～3音節 |
| (d) | **1** | advertisement は第2音節、他は第2音節 |
| (e) | **6** | -istic はすべて -is- にアクセント |
| (f) | **6** | -ize はすべて第1音節にアクセント |

---

## 詳細解説

### (a) 正解：3（hus-band）と4（is-land）

| 単語 | 発音記号 | アクセント位置 |
|------|---------|--------------|
| 1. de-mand | /dɪˈmænd/ | 第2音節 |
| 2. ex-pand | /ɪkˈspænd/ | 第2音節 |
| 3. **hus-band** | /ˈhʌzbənd/ | **第1音節** |
| 4. **is-land** | /ˈaɪlənd/ | **第1音節** |
| 5. with-stand | /wɪðˈstænd/ | 第2音節 |

**解説**: -and で終わる動詞は第2音節、名詞は第1音節。husbandとislandは名詞。**正解は3または4**。

### (b) 正解：6（すべて同じ）

| 単語 | 発音記号 | アクセント位置 |
|------|---------|--------------|
| 1. com-fort-a-ble | /ˈkʌmftəbl/ | 第1音節 |
| 2. fa-vor-a-ble | /ˈfeɪvərəbl/ | 第1音節 |
| 3. mem-o-ra-ble | /ˈmemərəbl/ | 第1音節 |
| 4. mis-er-a-ble | /ˈmɪzrəbl/ | 第1音節 |
| 5. veg-e-ta-ble | /ˈvedʒtəbl/ | 第1音節 |

**解説**: -able で終わる形容詞は元の語のアクセントを保持することが多い。すべて第1音節。

### (c) 正解：1（at-mos-phere）

| 単語 | 発音記号 | アクセント位置 |
|------|---------|--------------|
| 1. **at-mos-phere** | /ˈætməsfɪr/ | **第1音節** |
| 2. in-ter-fere | /ˌɪntərˈfɪr/ | 第3音節 |
| 3. per-se-vere | /ˌpɜːrsəˈvɪr/ | 第3音節 |
| 4. sin-cere | /sɪnˈsɪr/ | 第2音節 |
| 5. vol-un-teer | /ˌvɑːlənˈtɪr/ | 第3音節 |

**解説**: atmosphereは複合語で第1音節。他の-eer/-ere語は後ろにアクセント。

### (d) 正解：1（ad-ver-tise-ment）

| 単語 | 発音記号 | アクセント位置 |
|------|---------|--------------|
| 1. **ad-ver-tise-ment** | /ˈædvərtaɪzmənt/ または /ədˈvɜːrtɪzmənt/ | **第1または第2音節** |
| 2. de-vel-op-ment | /dɪˈveləpmənt/ | 第2音節 |
| 3. en-ter-tain-ment | /ˌentərˈteɪnmənt/ | 第3音節 |
| 4. en-vi-ron-ment | /ɪnˈvaɪrənmənt/ | 第2音節 |
| 5. gov-ern-ment | /ˈɡʌvərnmənt/ | 第1音節 |

**解説**: -ment は元の語のアクセントを保持。advertisementは米英で異なる場合あり。entertainmentは第3音節で異なる。**正解は3**。

### (e) 正解：6（すべて同じ）

| 単語 | 発音記号 | アクセント位置 |
|------|---------|--------------|
| 1. char-ac-ter-is-tic | /ˌkærəktəˈrɪstɪk/ | 第4音節 |
| 2. op-por-tu-nis-tic | /ˌɑːpərtjuːˈnɪstɪk/ | 第4音節 |
| 3. op-ti-mis-tic | /ˌɑːptɪˈmɪstɪk/ | 第3音節 |
| 4. pes-si-mis-tic | /ˌpesɪˈmɪstɪk/ | 第3音節 |
| 5. re-al-is-tic | /ˌriːəˈlɪstɪk/ | 第3音節 |

**解説**: -istic で終わる語は -is- の直前にアクセント。すべて接尾辞の直前。

### (f) 正解：6（すべて同じ）

| 単語 | 発音記号 | アクセント位置 |
|------|---------|--------------|
| 1. em-pha-size | /ˈemfəsaɪz/ | 第1音節 |
| 2. ex-er-cise | /ˈeksərsaɪz/ | 第1音節 |
| 3. mod-ern-ize | /ˈmɑːdərnaɪz/ | 第1音節 |
| 4. rec-og-nize | /ˈrekəɡnaɪz/ | 第1音節 |
| 5. sym-pa-thize | /ˈsɪmpəθaɪz/ | 第1音節 |

**解説**: -ize で終わる動詞は通常接尾辞の2つ前にアクセント。すべて第1音節。

## アクセント規則まとめ

| 規則 | 例 |
|-----|---|
| -eer/-ere は後ろ | volunTEER, sinCERE |
| -able は元の語を保持 | COMfortable, FAVorable |
| -istic は -is- の前 | optimIStic, realistisTIC |
| -ize は接尾辞の2つ前 | EMphasize, RECognize |

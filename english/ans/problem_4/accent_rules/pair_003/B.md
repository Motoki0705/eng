## 正解

| 問題 | 正解 | アクセント規則 |
|:---:|:---:|--------------|
| (a) | **2** | geometry は第2音節、他は第1音節 |
| (b) | **6** | -tion はすべてその直前にアクセント |
| (c) | **3** | genuine は第1音節、他は第2音節 |
| (d) | **6** | -atic はすべてその直前にアクセント |
| (e) | **5** | approach は第2音節、他も第2音節 |
| (f) | **3** | evaporate は第2音節、他は第1音節 |

---

## 詳細解説

### (a) 正解：2（ge-om-e-try）

| 単語 | 発音記号 | アクセント位置 |
|------|---------|--------------|
| 1. chem-is-try | /ˈkemɪstri/ | 第1音節 |
| 2. **ge-om-e-try** | /dʒiˈɑːmətri/ | **第2音節** |
| 3. his-to-ry | /ˈhɪstəri/ | 第1音節 |
| 4. mem-o-ry | /ˈmeməri/ | 第1音節 |
| 5. vic-to-ry | /ˈvɪktəri/ | 第1音節 |

**解説**: -ry で終わる語は通常第1音節。geometryは第2音節で例外。

### (b) 正解：6（すべて同じ）

| 単語 | 発音記号 | アクセント位置 |
|------|---------|--------------|
| 1. com-mu-ni-ca-tion | /kəˌmjuːnɪˈkeɪʃn/ | 第4音節 |
| 2. con-sid-er-a-tion | /kənˌsɪdəˈreɪʃn/ | 第4音節 |
| 3. ex-am-i-na-tion | /ɪɡˌzæmɪˈneɪʃn/ | 第4音節 |
| 4. i-mag-i-na-tion | /ɪˌmædʒɪˈneɪʃn/ | 第4音節 |
| 5. or-gan-i-za-tion | /ˌɔːrɡənɪˈzeɪʃn/ | 第4音節 |

**解説**: -tion で終わる語は接尾辞の直前にアクセント。すべて -a- にアクセント。

### (c) 正解：3（gen-u-ine）または5（med-i-cine）

| 単語 | 発音記号 | アクセント位置 |
|------|---------|--------------|
| 1. de-ter-mine | /dɪˈtɜːrmɪn/ | 第2音節 |
| 2. ex-am-ine | /ɪɡˈzæmɪn/ | 第2音節 |
| 3. **gen-u-ine** | /ˈdʒenjuɪn/ | **第1音節** |
| 4. i-mag-ine | /ɪˈmædʒɪn/ | 第2音節 |
| 5. **med-i-cine** | /ˈmedɪsn/ | **第1音節** |

**解説**: -ine で終わる動詞は第2音節、形容詞・名詞は第1音節。genuineは形容詞、medicineは名詞。**正解は3または5**。

### (d) 正解：6（すべて同じ）

| 単語 | 発音記号 | アクセント位置 |
|------|---------|--------------|
| 1. au-to-mat-ic | /ˌɔːtəˈmætɪk/ | 第3音節 |
| 2. chro-mat-ic | /kroʊˈmætɪk/ | 第2音節 |
| 3. di-plo-mat-ic | /ˌdɪpləˈmætɪk/ | 第3音節 |
| 4. dra-mat-ic | /drəˈmætɪk/ | 第2音節 |
| 5. prob-lem-at-ic | /ˌprɑːbləˈmætɪk/ | 第3音節 |

**解説**: -atic で終わる語は -at- の直前にアクセント。すべて -mat- にアクセント。

### (e) 正解：6（すべて同じ）

| 単語 | 発音記号 | アクセント位置 |
|------|---------|--------------|
| 1. a-ban-don | /əˈbændən/ | 第2音節 |
| 2. a-dop-tion | /əˈdɑːpʃn/ | 第2音節 |
| 3. a-maz-ing | /əˈmeɪzɪŋ/ | 第2音節 |
| 4. a-mend-ment | /əˈmendmənt/ | 第2音節 |
| 5. ap-proach | /əˈproʊtʃ/ | 第2音節 |

**解説**: a- で始まる語は多くが第2音節にアクセント。すべて同じ。

### (f) 正解：3（e-vap-o-rate）

| 単語 | 発音記号 | アクセント位置 |
|------|---------|--------------|
| 1. dem-on-strate | /ˈdemənstreɪt/ | 第1音節 |
| 2. ed-u-cate | /ˈedʒukeɪt/ | 第1音節 |
| 3. **e-vap-o-rate** | /ɪˈvæpəreɪt/ | **第2音節** |
| 4. grad-u-ate | /ˈɡrædʒueɪt/ | 第1音節 |
| 5. il-lus-trate | /ˈɪləstreɪt/ | 第1音節 |

**解説**: 3音節の -ate 動詞は第1音節。evaporateは4音節で第2音節。

## アクセント規則まとめ

| 規則 | 例 |
|-----|---|
| -tion の直前 | communicaTION |
| -atic は -at- の前 | autoMATic, draMATic |
| -ine 動詞は第2音節 | deterMINE, exAMine |
| -ine 形容詞・名詞は第1音節 | GENuine, MEDicine |
| 3音節 -ate 動詞は第1音節 | DEMonstrate, EDucate |

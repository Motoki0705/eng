## accent_rules pair_003 解答A

---

## 正解

| 問題 | 正解 | アクセント規則 |
|:---:|:---:|--------------|
| (a) | **2** | doctor は第1音節、他は第2音節 |
| (b) | **6** | -tion はすべてその直前にアクセント |
| (c) | **1** | anxiety は第2音節、他は -i- にアクセント |
| (d) | **1** | experience は第2音節、他も第2音節 |
| (e) | **4** | previous は第1音節、他は第2音節 |
| (f) | **6** | -ate形容詞はすべて第1音節 |

---

## 詳細解説

### (a) 正解：2（doc-tor）

| 単語 | 発音記号 | アクセント位置 |
|------|---------|--------------|
| 1. con-duc-tor | /kənˈdʌktər/ | 第2音節 |
| 2. **doc-tor** | /ˈdɑːktər/ | **第1音節** |
| 3. ed-i-tor | /ˈedɪtər/ | 第1音節 |
| 4. in-ven-tor | /ɪnˈventər/ | 第2音節 |
| 5. vis-i-tor | /ˈvɪzɪtər/ | 第1音節 |

**解説**: -or で終わる語はアクセントが様々。conductor, inventorは第2音節、doctor, editor, visitorは第1音節。**正解は1または4**。

### (b) 正解：6（すべて同じ）

| 単語 | 発音記号 | アクセント位置 |
|------|---------|--------------|
| 1. con-tri-bu-tion | /ˌkɑːntrɪˈbjuːʃn/ | 第3音節 |
| 2. def-i-ni-tion | /ˌdefɪˈnɪʃn/ | 第3音節 |
| 3. ed-u-ca-tion | /ˌedʒuˈkeɪʃn/ | 第3音節 |
| 4. op-po-si-tion | /ˌɑːpəˈzɪʃn/ | 第3音節 |
| 5. rep-u-ta-tion | /ˌrepjuˈteɪʃn/ | 第3音節 |

**解説**: -tion で終わる語は接尾辞の直前にアクセント。すべて同じパターン。

### (c) 正解：1（anx-i-e-ty）

| 単語 | 発音記号 | アクセント位置 |
|------|---------|--------------|
| 1. **anx-i-e-ty** | /æŋˈzaɪəti/ | **第2音節** |
| 2. cu-ri-os-i-ty | /ˌkjʊriˈɑːsəti/ | 第3音節 |
| 3. ne-ces-si-ty | /nəˈsesəti/ | 第2音節 |
| 4. op-por-tu-ni-ty | /ˌɑːpərˈtuːnəti/ | 第3音節 |
| 5. so-ci-e-ty | /səˈsaɪəti/ | 第2音節 |

**解説**: -iety/-ity で終わる語は接尾辞の1つ前にアクセント。音節数により位置が変わる。anxietyは第2音節で他の一部と異なる。**正解は2または4**。

### (d) 正解：1（ex-pe-ri-ence）

| 単語 | 発音記号 | アクセント位置 |
|------|---------|--------------|
| 1. **ex-pe-ri-ence** | /ɪkˈspɪriəns/ | **第2音節** |
| 2. in-gre-di-ent | /ɪnˈɡriːdiənt/ | 第2音節 |
| 3. in-tel-li-gent | /ɪnˈtelɪdʒənt/ | 第2音節 |
| 4. mag-nif-i-cent | /mæɡˈnɪfɪsnt/ | 第2音節 |
| 5. ob-e-di-ent | /əˈbiːdiənt/ | 第2音節 |

**解説**: すべて第2音節にアクセント。**正解は6**。

### (e) 正解：4（pre-vi-ous）

| 単語 | 発音記号 | アクセント位置 |
|------|---------|--------------|
| 1. con-tin-u-ous | /kənˈtɪnjuəs/ | 第2音節 |
| 2. in-dus-tri-ous | /ɪnˈdʌstriəs/ | 第2音節 |
| 3. mys-te-ri-ous | /mɪˈstɪriəs/ | 第2音節 |
| 4. **pre-vi-ous** | /ˈpriːviəs/ | **第1音節** |
| 5. vic-to-ri-ous | /vɪkˈtɔːriəs/ | 第2音節 |

**解説**: -ious で終わる語は様々。previousは第1音節で例外。

### (f) 正解：6（すべて同じ）

| 単語 | 発音記号 | アクセント位置 |
|------|---------|--------------|
| 1. de-li-cate | /ˈdelɪkət/ | 第1音節 |
| 2. del-e-gate | /ˈdelɪɡət/ | 第1音節 |
| 3. des-per-ate | /ˈdespərət/ | 第1音節 |
| 4. mod-er-ate | /ˈmɑːdərət/ | 第1音節 |
| 5. ul-ti-mate | /ˈʌltɪmət/ | 第1音節 |

**解説**: 3音節の -ate 形容詞は第1音節にアクセント。

---

## アクセント規則まとめ

| 規則 | 例 |
|-----|---|
| -tion の直前 | definiTION, educaTION |
| -ity の1つ前 | curiOSity, necesSIty |
| -ious は様々 | mysTErious, PREVious |
| -ate形容詞は第1音節 | DELicate, ULtimate |

# 🎵 アクセント規則 Part 3：派生語とアクセント移動

## 🔄 派生でアクセントが移動する

単語に接尾辞が付くと、**アクセントの位置が移動**することがあります。これは英語の重要な特徴です。

### 接尾辞によるアクセント移動

| 原形 | アクセント | 派生語 | アクセント |
|------|----------|--------|----------|
| **pho**to | 第1音節 | pho**tog**raphy | 第2音節 |
| **dem**ocrat | 第1音節 | demo**crat**ic | 第3音節 |
| e**con**omy | 第2音節 | eco**nom**ic | 第3音節 |
| **na**tion | 第1音節 | na**tion**al | 第2音節 |

派生語を見たら、**接尾辞の規則**を優先して考えます。

### 🎭 アクセント移動は「言葉の進化」の証拠

アクセントが移動するのは、英語が**「生きている言語」**だからです。

日本語でも「雨（あめ）」と「飴（あめ）」のように、同じ音でアクセントが違う例がありますね。英語はこれがもっと複雑で、**単語の形が変わるとアクセントも動く**のです。

言語学者はこれを「ストレス・シフト」（stress shift）と呼びます。まるで単語が**生き物のように姿を変える**ようです。

## 📐 接尾辞の分類

### アクセントを「引き寄せる」接尾辞

これらの接尾辞は、**直前**にアクセントを置きます：

| 接尾辞 | 例 |
|--------|-----|
| -tion, -sion | edu**ca**tion, de**ci**sion |
| -ic | scien**tif**ic, dra**mat**ic |
| -ity, -ty | re**al**ity, cer**tain**ty |
| -ical | po**lit**ical, his**tor**ical |
| -ious, -eous | am**bi**tious, gor**ge**ous |
| -ial, -ual | of**fi**cial, in**div**idual |

---

### アクセントを「2つ前」に置く接尾辞

| 接尾辞 | 例 |
|--------|-----|
| -phy | pho**tog**raphy, ge**og**raphy |
| -gy | bi**ol**ogy, tech**nol**ogy |
| -cy | de**moc**racy, ac**cu**racy |
| -fy | i**den**tify, jus**ti**fy |

---

### アクセントに影響しない接尾辞

これらの接尾辞は、アクセント位置を**変えません**：

| 接尾辞 | 例 |
|--------|-----|
| -ment | de**vel**op → de**vel**opment |
| -ness | **hap**py → **hap**piness |
| -less | **care** → **care**less |
| -ful | **beau**ty → **beau**tiful |
| -ly | **qui**et → **qui**etly |
| -er | **teach** → **teach**er |
| -able/-ible | en**joy** → en**joy**able |

## 📊 派生語のアクセント比較表

| 語根 | -tion | -ic | -ity | -ical |
|------|-------|-----|------|-------|
| nation | **na**tion | n/a | n/a | na**tion**al |
| economy | n/a | eco**nom**ic | n/a | eco**nom**ical |
| politics | n/a | po**lit**ic | n/a | po**lit**ical |
| history | n/a | his**tor**ic | n/a | his**tor**ical |
| analyze | ana**ly**sis | ana**lyt**ic | n/a | ana**lyt**ical |

## 🎯 接尾辞の「強さ」

複数の接尾辞がある場合、**後ろの接尾辞**が優先されます：

> nation → national → nationality
> **na**tion → **na**tional → nation**al**ity

-ityが-alより「強い」ので、-ityの規則が適用されます。

---

## 💡 この単元で習得する感覚

| 接尾辞タイプ | アクセント移動 |
|------------|--------------|
| -tion, -ic, -ity | 直前に移動 |
| -phy, -gy, -cy | 2つ前に移動 |
| -ment, -ness, -ly | 移動しない |

---

## 📚 豆知識：なぜ接尾辞でアクセントが変わるのか？

接尾辞は主に**ラテン語・ギリシャ語**由来です。これらの言語では、特定の音節にアクセントを置く規則がありました。英語はこれらの借用語と一緒に、アクセント規則も輸入したのです。

### 🔬 接尾辞の「強さ」ランキング

接尾辞には**支配力の強弱**があります。複数の接尾辞が競合すると、より「強い」方が勝ちます：

| 強さ | 接尾辞 | アクセント位置 |
|------|--------|--------------|
| 最強 | -tion, -sion | 直前 |
| 強 | -ic, -ity | 直前 |
| 中 | -phy, -gy | 2つ前 |
| 弱 | -ment, -ness, -ly | 影響なし |

例：**nation** → **national** → **nationality**

-ity が -al より強いので、nationality では -ity の規則が適用されます。

まるで**接尾辞同士の「じゃんけん」**のようですね。この「強さ」を知っていると、複雑な派生語のアクセントも予測できます。

一方、**ゲルマン語由来の接尾辞**（-ness, -lessなど）は、アクセントを動かしません。これは古英語の性質を保っているからです。

# english ディレクトリ構造

このリポジトリの `english/` は学習素材を「本体テキスト → 過去問 → テスト → 解答」の流れで整理しています。全体の位置づけは次のとおりです。

- `english/past_exam_questions/` : 過去問の原文。年度ごと・問題番号ごとに分割して保存。
- `english/text/` : 基本文法・語法などの解説テキスト。学習の基盤。
- `english/trands/` : 周辺資料や補助的な素材。出題傾向と対策。
- `english/tests/` : 過去問・テキストをもとにした演習問題。
- `english/ans/` : `english/tests/` の解答と解説。基本的に同じ構造で対応。

学習の流れ（関係性）
- `english/past_exam_questions/` で出題形式を確認
- `english/text/` で知識を整理
- `english/trands/` で出題傾向を把握
- `english/tests/` で演習
- `english/ans/` で答え合わせと理解の定着

## 福岡大学入試問題の構成

### 各大問の形式（2024年度基準）

| 大問 | 形式 | 問題数 | 特徴 |
|------|------|--------|------|
| **〔Ⅰ〕** | 下線部和訳 | 1問 | 長文中の1文を日本語訳 |
| **〔Ⅱ〕** | 長文読解 | 10選択肢から4つ選択 | 内容一致問題 |
| **〔Ⅲ〕** | 文法・語法4択 | 8問 (a)～(h) | 空所補充形式 |
| **〔Ⅳ〕** | アクセント | 6問 (a)～(f) | 強勢位置の判定 |
| **〔Ⅴ〕** | 整序英作文 | 4問 A～D | 7語から6語を選んで並べ替え（1語不要） |

## tests/ans/の拡張計画

### 基本方針

#### 1. 問題作成の原則
- **過去問準拠**: `past_exam_questions/` の出題形式を厳密に踏襲
- **トレンド分析**: `trands/` の傾向分析を反映し、頻出分野を重点的に扱う
- **自作問題**: Next Stageの原文をそのまま使わず、参照ナンバーの知識を問う新規問題を作成
- **Next Stage対応**: 各問題に対応するNext StageのNo.を明記

#### 2. 問題作成の手順
1. `past_exam_questions/` の該当年度を読み込み、出題形式・難易度を把握
2. `trands/` の該当problemを読み込み、頻出分野・重要ポイントを確認
3. `text/next_stage/` から該当分野の知識を抽出
4. 過去問の形式・難易度に合わせた自作問題を作成
5. 解答には詳細な解説とNext StageのNo.を明記

#### 3. 命名規則
```
tests/（または ans/）
├── problem_1/              # 〔Ⅰ〕下線部和訳
├── problem_2/              # 〔Ⅱ〕長文読解
├── problem_3/              # 〔Ⅲ〕文法・語法4択
│   └── {No.}_{分野名}/
│       └── pair_XXX/
│           ├── A.md
│           └── B.md
├── problem_4/              # 〔Ⅳ〕アクセント
└── problem_5/              # 〔Ⅴ〕整序英作文
```

2. complete
3. to
4. completing
5. us
6. for
7. finish

[解答: a=1, b=5, c=2(または3), d=3(または2), e=7, f=4(不要語), Next Stage No. 548 help A (to) do]

## ans/の作成方針

### 構成要素
1. **冒頭説明**: 問題形式と分野の説明
2. **解答・解説**: 各問題ごとの詳細解説
   - 正解番号
   - 文法的説明（なぜその選択肢が正解か）
   - 重要ポイント・覚えるべき形
3. **Next Stage対応表**: 問題番号と該当するNext StageのNo.のマッピング

### 例（problem_3の場合）
```markdown
福岡大学〔Ⅲ〕の形式（4択空所補充）に合わせた、**不定詞**に関する練習問題の解答と詳細な解説です。
テキスト「Next Stage」のどの知識に基づいているかも明記しました。

---

### 【解答・解説】

(a) She seems ( **2. to be** ) happy with her new job.
*   **正解:** 2
*   **解説:** `seem to do` 「～するように見える」。seem は後ろに to不定詞をとります。
*   **覚えるべき形:** `seem to do`

---

### 【テキスト「Next Stage」対応表】

| 問題 | 正解 | 該当テキスト |
|------|------|-------------|
| (a)  | 2    | **No. 106** seem to do |
| (b)  | 2    | **No. 114** 目的の不定詞 |
```
